# Goviel

## Project Info

Goviel Application using React, Vite, Shadcn UI, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
